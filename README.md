## UMBRACO CHALLENGE

#### using Umbraco version 12.3.3 and dotnet version 7

### Installing Umbraco

```bash
  dotnet new install Umbraco.Templates
```

### Running the program

```bash
  cd Umbraco-Challenge
  dotnet watch run
```
